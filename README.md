✨ Tính năng nổi bật
Tạo câu hỏi theo chủ đề: Chỉ cần nhập một chủ đề, AI sẽ tự động tạo ra một bộ câu hỏi trắc nghiệm hoặc tự luận liên quan.
Tạo câu hỏi theo mẫu: Cung cấp một câu hỏi mẫu, AI sẽ dựa vào đó để tạo ra các câu hỏi khác với cùng cấu trúc.
Phân loại độ khó: Tự động gán nhãn độ khó (Dễ, Trung bình, Khó) cho từng câu hỏi, giúp người dùng dễ dàng lựa chọn và ôn tập.
Giao diện hiện đại: Được xây dựng bằng ReactJS và TailwindCSS, mang lại trải nghiệm người dùng mượt mà và trực quan.
🚀 Công nghệ sử dụng
Framework: ReactJS
Styling: TailwindCSS
Ngôn ngữ: JavaScript
API: Google Gemini API
Quản lý gói: NPM
